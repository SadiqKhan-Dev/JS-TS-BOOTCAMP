Q1) What is DOM?

DOM stands for Document Object Model

It is a programming interface provided bu the browser that allows developers to interact
with and manipulate HTML documents.

it represents the structure of the document as a tree of objects.

Each element attribute and text in the HTML document becomes a node in the DOM.

Javascript is commonly used to manipulate the DOM to change content style and structure dynamically.

Q3) What is BOM?

BOM stand for Browser Objec Model.

It allows Javascript to interact with the browser outside of the HTML document.

  • window
  • navigator
  • location
  • history
  • screen
  • alert, confirm, prompt
